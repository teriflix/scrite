# Scriptalay

Scriptalay is a small online library of screenplays in Scrite format. This
library is visible in the [Home Screen](./quickstart.md#home-screen) when you
use the keyboard shortcut `Ctrl+Shift+O`.

<img src="../images/scriptalay/001-scriptalay.jpg" width="100%"/>

You can select a script to see its poster and some quick meta-data. Clicking on
the `Open` button downloads and opens that screenplay for you.

<img src="../images/scriptalay/002-scriptalay.jpg" width="100%"/>

You can switch between various tabs, generate reports, export to other formats,
and in general explore all features of Scrite.

If you would like to contribute your screenplay to this library, please do not
hesitate to reach out on `support@scrite.io`.
